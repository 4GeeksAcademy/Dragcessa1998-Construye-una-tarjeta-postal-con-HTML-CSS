# Mi postal digital

Ejercicio de HTML y CSS para 4Geeks Academy. El objetivo es construir una
postal centrada en pantalla usando estructura HTML5, un archivo CSS enlazado,
una fuente de Google Fonts, cajas con márgenes y padding, y un formulario
sencillo.

## Archivos principales

- `src/index.html`: versión con la estructura esperada por el ejercicio.
- `src/style.css`: estilos de la postal.
- `index.html` y `styles.css`: copia preparada para abrir o publicar la postal
  directamente desde la raíz del repositorio.

## Cómo verlo

Abre `index.html` en el navegador o publica el repositorio con GitHub Pages.

## Estado

La postal incluye:

- Estructura HTML5.
- CSS externo enlazado.
- Logo de 4Geeks Academy.
- Fuente importada desde Google Fonts.
- Maquetación con cajas, margen, padding y `overflow`.
- Formulario simple con nombre, email y mensaje.
